Application bureau

Ce projet pour l'entreprise fictive AngerScoot s'est déroulé dans le cadre d'un AP durant ma première année de BTS. Il a consisté à créer une application backoffice en C# connectée à une BDD MySQL.

Description

Ce projet s'est déroulé sur deux mois dans le cadre d'un AP. Il a été réalisé en équipe de 2. Pour ce projet, les travaux ont été séparés en deux. Je me suis occupé de l'application backoffice, de la mise en place des serveurs WEB et BDD et de la modélisation de la BDD. Mon coéquipier a pris en charge l'application WEB et la charte graphique.

Mes travaux ont consisté à créer une application en C# en utilisant le Framework WindowsForms pour l'entreprise fictive AngerScoot. Cette application a pour but de permettre aux administrateurs de gérer le parc de véhicule et les employés de l'entreprise. Cette application utilise une Base de Données MySQL pour stocker les informations nécessaires au bon fonctionnement de l'entreprise.

Travaux réalisés

1. Création de la Base de Données

La Base de Données a été créée à l'aide de WinDesign. Le modèle a été construit en nous appuyant sur les informations fournies en entré de l'AP. Nous l'avons ensuite déployé sur un serveur hébergé sur une machine virtuelle Linux.

2. Développement de l'application

L'application à été développée en C# avec WindowsForms. Elle comprend une dizaine d'écrans.

Compétences aquises

Ce projet m'a permis d'apprendre à développer une application complète en partant d'un besoin formulé par un client. J'ai aussi appris à respecter les délais pour finir un projet et à administrer des serveur WEB et BDD pour les sécuriser.
